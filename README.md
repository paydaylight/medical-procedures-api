### Medical Procedures API
ruby version: 2.5.1
rails version: 5.2.4

# getting started: 
 * bundle install
 * yarn install
 * rails db:create
 * rails db:migrate
 * rails db:seed
 
# running tests:
 * bundle exec rspec

# generating documentation:
 * rake api:docs
 * Then the documentation will be accessible [here](/documentation.html)

Postman collection is [here](/docs/api/MEDICAL%20PROCEDURES%20API.postman_collection.json)
